In Go, you can convert a byte array to a string using the `string()` function. This function takes a byte array as an argument and returns a string. You can also use the `fmt.Sprintf()` function to convert a byte array to a string.`

## Convert byte array to string using string() <SectionLink id="convert-byte-array-to-string-using-string" />

The `string()` function takes a byte array as an argument and returns a string. The following example shows how to convert a byte array to a string using the `string()` function.

```go title="main.go"
package main

import "fmt"

func main() {
    byteArray := []byte{0x68, 0x65, 0x6c, 0x6c, 0x6f}
    str := string(byteArray)
    fmt.Println(str)
}
```

The output of the above program is:

```bash title="Output"
hello
```

```go title="main.go"
package main

import "fmt"

func main() {
	byteArray := []byte{'h', 'e', 'l', 'l', 'o'}
	str := string(byteArray[:])
	fmt.Println(str)
}

```

## Convert byte array to string using fmt.Sprintf() <SectionLink id="convert-byte-array-to-string-using-fmt-sprintf" />

The `fmt.Sprintf()` function takes a format string and a list of arguments as arguments and returns a string. The following example shows how to convert a byte array to a string using the `fmt.Sprintf()` function.

```go title="main.go"
package main

import "fmt"

func main() {
    byteArray := []byte{0x68, 0x65, 0x6c, 0x6c, 0x6f}
    str := fmt.Sprintf("%s", byteArray)
    fmt.Println(str)
}
```

The output of the above program is:

```bash title="Output"
hello
```

## Convert byte array to string using a slice of bytes <SectionLink id="convert-byte-array-to-string-using-a-slice-of-bytes" />

You can also convert a byte array to a string by slicing the byte array.

```go title="main.go"
package main

import "fmt"

func main() {
    byteArray := []byte{0x68, 0x65, 0x6c, 0x6c, 0x6f, 0x20, 0x77, 0x6f, 0x72, 0x6c, 0x64}
    str := string(byteArray[6:])
    fmt.Println(str)
}
```

The output of the above program is:

```bash title="Output"
world
```
